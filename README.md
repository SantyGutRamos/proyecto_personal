El nombre de mi pagina es Academia Canina  
URL:https://santygutramos.github.io/proyecto_personal/
Trata sobre el cuidado que le damos a los caninos tomando encuenta algunas razas,con ayuda de una calculadora para saber su consumo de comida,informacion especifica del animal mas sus cuidados especificos y una pequeñas recomendaciones para su adiestramiento.
En mi codigo se uso mayormente html,css,Js y en pequeña media python que lo utilize para conectarlo a una base de datos(SQL).
